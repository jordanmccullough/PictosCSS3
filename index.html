Pictos & CSS3

Friendly Pictogram Implementation

Benefits

Avoiding unnecessary characters, used only for stylistic/visual cues, within markup is an important aspect of design. Not only does this approach keep visual elements in CSS where it belongs, but prevents polluting markup for accessibility and crawlability.

Caveats

Line-height isn't applied as an inline character for the representative character would.
